# csc-views

This module adds customization to various views in the CSC website at csc.virginia.edu.

The modifications it makes are:
## Event Row Teasers View
* **View:** events
* **Display:** block_2
* **Feature:** Takes first slug from path and promotes any upcoming event tagged with that string to the top of the list.
* **Example:** on any page with path /research/ it will promote events tagged with research to the top of the list.
